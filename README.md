📌 Self-Pruning Neural Network

📖 Description

A Self-Pruning Neural Network is a deep learning model that automatically removes unnecessary weights during training. It improves efficiency, reduces model size, and maintains accuracy using sparsity constraints.

🚀 Features

Automatic weight pruning during training
Reduced model complexity
Improved efficiency and faster inference
Maintains accuracy with fewer parameters

🛠️ Installation

pip install -r requirements.txt

Or install manually:

pip install torch torchvision matplotlib
